# Identidad visual municipal para proyectos DIA

Esta guía debe aplicarse a las pantallas de Applaza y servir como referencia para futuros proyectos de DIA.

## Marca

- Archivo canónico: `public/logo-municipal.png`.
- Favicon de Next.js: `app/icon.png`.
- No deformar, recolorear, rotar ni recortar el símbolo.
- Mantener fondo blanco y espacio libre alrededor de la marca.

## Paleta extraída del archivo oficial

| Token | Hex | Uso |
|---|---|---|
| Azul municipal | `#0166FF` | Acciones principales, navegación y foco |
| Celeste municipal | `#2DB0FF` | Gradientes, información y estados secundarios |
| Amarillo municipal | `#F4DC00` | Acentos y selección; no usar para texto pequeño |
| Blanco | `#FFFFFF` | Fondo del logotipo y superficies |

Los colores semánticos de éxito, advertencia y error pueden conservar verde, naranja y rojo para no perder accesibilidad. La identidad municipal se aplica a navegación, botones, enlaces, foco, encabezados y fondos institucionales.
